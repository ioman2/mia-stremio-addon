# M.I.A. RO Stremio Subtitle Addon

Addon static pentru subtitrările românești M.I.A. S01, IMDb `tt33253070`.

## Render

Build Command:

```bash
npm install
```

Start Command:

```bash
npm start
```

Environment Variable:

```text
PUBLIC_URL=https://stremio-regielive.onrender.com
```

Instalare în Stremio:

```text
https://stremio-regielive.onrender.com/manifest.json
```

Test episod 1:

```text
https://stremio-regielive.onrender.com/subtitles/series/tt33253070:1:1.json
```
